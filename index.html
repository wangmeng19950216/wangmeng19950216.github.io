iview Table 表格的 render 函数

render 函数传入两个参数，第一个是 h，第二个是对象，包含 row、column 和 index，分别指当前单元格数据，当前列数据，当前是第几行。

iview表格的render函数作用是自定义渲染当前列，权限高于key，所以使用了render函数，那么表格的key值就无效了。

render:(h,params) => {
return h(“ 定义的元素 “，{ 元素的性质 }，” 元素的内容”/[元素的内容])
}

render: (h, params) => {
Params.row 就是当前行
return h(‘div’, [
h(‘Icon’, {
props: {
type: ‘person’
}
}),
h(‘strong’, params.row.name)
]);
H(‘button’,{
Props:{
Type:’’,
Size:’’
},
on:{
Click:{
创建的button按钮的点击事件
}
}

})

}

对于render函数而言，我们可以将h看作一个创建元素的字段,创建好这个元素之后,需要添加样式或者做其他操作,后面跟着一个对象(里面是属性和方法)或者数组(该元素的子元素)就行。涉及到的几个属性无外乎props,style，on，attrs
